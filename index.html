Task-manager

This application helps manage tasks. The application logic: to organize tasks, a board with cards is used, which are distributed by type: backlog, in progress, review or done. Users can create cards, fill in the description and edit the text, change the position of the block in the list, move it through the lists, and delete a card or all done-cards. Also in the card, you can assign a user to participants in each task and set a priority of completion. There is a convenient card filter by various parameters, as well as a task search. For the convenience, you can change the color theme (from dark to light).The following technologies were used in the project: HTML5, SCSS, JavaScript, Bootstrap.   The project was developed in a team in which I was appointed a team leader and, in addition to writing code, distributed tasks among team members, made code reviews and approved pull requests on github.
